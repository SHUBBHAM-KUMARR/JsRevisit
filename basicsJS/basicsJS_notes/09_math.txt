in maths we have different method husing which we can make differrent calculations

| Method          | Description                     | Example           | Output |
| --------------- | ------------------------------- | ----------------- | ------ |
| `Math.abs(x)`   | Absolute value                  | `Math.abs(-9)`    | `9`    |
| `Math.ceil(x)`  | Round **up** to nearest integer | `Math.ceil(4.2)`  | `5`    |
| `Math.floor(x)` | Round **down**                  | `Math.floor(4.7)` | `4`    |
| `Math.round(x)` | Round to nearest integer        | `Math.round(4.5)` | `5`    |
| `Math.trunc(x)` | Remove decimal part (truncate)  | `Math.trunc(4.9)` | `4`    |
